# randomProject

## 🛠 Technology Stack

### Frontend
- **React** (via Vite) → For building the user interface  
- **JavaScript** → Logic & interactivity  
- Dev server runs on **http://localhost:5175/**

### Backend
- **Node.js** → JavaScript runtime  
- **Express** → Web framework to handle API routes  
- **CORS** → Allows frontend to access backend  
- Backend server runs on **http://localhost:3000/**
